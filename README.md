# To-Do List Application

## 📌 Project Description

The **To-Do List Application** is a simple command-line application developed using Python. It helps users manage their daily tasks by allowing them to add, view, mark as completed, and delete tasks. The application stores tasks in a JSON file so that the data remains available even after the program is closed.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Save tasks automatically using JSON
* User-friendly menu-driven interface
* Input validation and error handling

---

## 🛠 Technologies Used

* Python 3
* JSON
* OS Module
